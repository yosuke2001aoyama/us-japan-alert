# JPUS Alert — 米国・日米関係速報

日米の主要報道、米政府の公式発表、要人発信を横断し、日本関連と重大案件を抽出する公開ダッシュボードです。

## できること

- White House、国務省、国防総省、Congress.gov、首相官邸・外務省などの一次情報を優先取得
- Truth Socialの公開APIから @realDonaldTrump の発信を取得
- Google News RSSを使い、日米の主要紙・通信・テレビ・政治メディアを横断
- 日本関連、一次情報、重大度、分野で自動分類
- 同一URLの重複排除、検索、絞り込み、CSV出力
- GitHub Actionsで15分ごとに収集し、新着の日本関連または重大度80以上をGitHub Issueで通知
- 通知Issueはリポジトリ所有者に自動割当されるため、GitHubの通常設定でメール通知も届く
- 一つの取得元が落ちても、他の監視を継続

## ローカル起動

```bash
npm ci
npm run dev
```

## 通知（Secrets不要）

監視にはGitHub Actionsが自動発行する `GITHUB_TOKEN` だけを使います。APIキーやメールアドレスをActions Secretsへ登録する必要はありません。

重要速報を検知すると、担当者付きのIssueが作られます。GitHubのメール通知を受けるには、アカウントの通知設定で「Email」を有効にしてください。リポジトリをWatchしておくと、担当者通知以外の更新も追いやすくなります。

初回実行は既存記事を基準データとして保存し、過去記事の大量通知は行いません。2回目以降に新しく検知した案件だけを通知します。

## 運用上の注意

「全メディア」の完全取得を保証するものではありません。公開RSS、検索RSS、公開APIで取得可能な範囲を広く監視し、ペイウォール記事は見出しと原文リンクを扱います。スクレイピング回避、出典明示、原文確認を前提としています。各提供元の仕様変更や利用規約変更に応じて取得経路を更新してください。
